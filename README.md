## Project Title
Airbnb Listings Market Analysis: Pricing, Availability, and Neighborhood Trends

## Tasks Completed
- Task 2: Exploratory Data Analysis
- Task 3: Data Visualization

## Project Overview
This project analyzes Airbnb listings data to understand pricing, availability, neighborhood trends, room type patterns, review activity, and host behavior.

The project was completed using Python in Jupyter Notebook. The analysis includes data cleaning, exploratory data analysis, feature creation, outlier handling, grouped analysis, and multiple visualizations.

## Dataset
The dataset used is Airbnb listings data from Inside Airbnb.

Main columns used:
- Listing ID
- Host information
- Neighborhood
- Room type
- Price
- Minimum nights
- Number of reviews
- Availability
- Reviews per month

## Key Insights
- Airbnb prices vary strongly by room type and neighborhood.
- Entire home/apartment listings usually have different pricing behavior compared with private rooms.
- Some neighborhoods have more listings, which may indicate stronger competition.
- Availability and review activity help reveal listing performance patterns.
- Data cleaning was important because some columns were fully empty and some listings had missing or invalid price values.
